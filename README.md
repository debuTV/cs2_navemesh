# cs2_navemesh
一个为npc建立导航网格的地图脚本，网格通过脚本生成后，需手动保存为静态文件，下次运行时自动加载。

具体教程请查看WIKI页面。

# 基准测试(v2.0 beta版本)
```
        //地图：nuke
        //起点：警家，终点：匪家
        //多边形总数：649跳点数：82
        //100次A*           42ms
        //100次funnelPath   55ms-42=13ms
        //100次50fixHeight    106ms-55=51ms
        //100次200fixHeight    70ms-55=15ms
```
# 注意事项
脚本具体参数需要手动调试，这会占用大量时间。

如果有疑问或者代码有bug，可以提出issues，但目前我没有什么动力进行更新，除非是特别恶劣的bug。

# cs2_navemesh
一个为npc建立导航网格的地图脚本，网格通过脚本生成后，需手动保存为静态文件，下次运行时自动加载。

具体教程请查看WIKI页面。

# 基准测试(v1.1版本)
```
        多边形总数：1025 跳点数：162
        100次A*           68ms
        100次funnelPath   6ms
        100次fixHeight    8ms
```
# 注意事项
脚本具体参数需要手动调试，这会占用大量时间。

如果有疑问或者代码有bug，可以提出issues，但目前我没有什么动力进行更新，除非是特别恶劣的bug。
